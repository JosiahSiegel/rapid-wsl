# [prime-reportstream](https://github.com/CDCgov/prime-reportstream)

```sh
d="Ubuntu"; # Distro name
m="generic"; # Module name

./slm init -d $d -m $m;
./slm status -d $d -m $m;
./slm backup -d $d -m $m;
./slm down -d $d -m $m;
./slm up -d $d -m $m;
./slm destroy -d $d -m $m;
./slm restore -d $d -m $m;
./slm test -d $d -m $m;

```
